## npm に同梱されている npx を利用することによる利点
### npm run
- package.jsonのscriptsに記載されたコマンドのみ実行可能
- プロジェクトにインストールされたバージョン固定
### npx
- 任意のnpmパッケージを指定して実行可能
- ローカルにすでにインストールされたパッケージでも、まだインストールされていないパッケージも指定可能
- ローカル優先だが、ない場合は一時的にキャッシュにインストールして実行可能

### 利点
- バージョン指定で実行できる
- 定期的に使わないようなパッケージを一時的にインストールして使用可能(使用後に削除)
- ローカルにインストールされているが、package.jsonに追加されていないものも実行可能(scriptsに追加不要)
