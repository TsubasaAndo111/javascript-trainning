# エクスポート元の関数・クラスをエディタのリファクタ機能で名前変更した際、インポート側で名前変更がどう追随されるか確認しなさい。

- 10.3(Nodeのモジュール方式)
エクスポートするモジュールのexport部分が以下のように変更された。
以下は、helloという関数をhello1にしたケース。
module.exports = {
    hello: hello1,
    Counter
};

- 10.4(ES6のモジュール方式)
1. デフォルトエクスポート
インポート側で名前を自由に設定できるため、追随されない。
2. 名前変更を伴うインポート、再エクスポート
追随された。
